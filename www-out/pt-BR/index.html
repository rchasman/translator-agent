translator-agent

A web foi feita para todos. Agora fala com todos.

A transcriação por IA transforma seu site em 71 versões locais. Não é tradução — é reescrita para cada cultura. Cada expressão adaptada, cada piada acertada, cada sistema de escrita contemplado.

5 bilhões de pessoas estão online. 60% da web está em inglês. A barreira nunca foi técnica — foi econômica. Localização profissional custa R$ 50.000 e leva um mês. Por isso a maioria das empresas lança em apenas um idioma e torce para dar certo.

A IA eliminou essa barreira

Não é tradução automática. É transcriação. O modelo sabe que "nossos hamsters estão trabalhando nisso" não funciona no Japão — e reescreve como uma piada culturalmente adequada. Sabe que R$ 250,00/mês vira R$ 250,00/mês. Sabe que alemão precisa de 30% mais espaço. Sabe que árabe inverte toda a página e o tailandês não usa espaços entre as palavras.

Aponte para sua build. Receba todos os idiomas.

bunx translator-agent -s ./dist -l all

Arquivos HTML recebem atributos lang e dir automaticamente. Cada idioma ganha CSS com a fonte, altura de linha e direção de texto corretos. Um middleware do Vercel detecta o Accept-Language sozinho. Faça o deploy uma vez, sirva o mundo todo.
