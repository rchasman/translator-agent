translator-agent

Ang internet ay para sa lahat. Ngayon, nagsasalita na ito sa lahat.

AI transcreation ang gagawing 71 lokal na bersyon ng iyong site. Hindi basta pagsasalin—bagkus, muling pagsulat para sa bawat kultura. Bawat idyoma ang inaangkop, bawat biro ay tumatagos, bawat sistema ng pagsulat ay hinaharap nang tama.

5 bilyong tao ang nakakapasok sa internet. 60% ng web ay nasa Ingles. Hindi teknikal ang hadlang—panlipunan, ito. Ang propesyonal na lokalisisyon ay nagkakahalaga ng ₱500,000 at tumatagal ng isang buwan. Kaya karamihan ng mga kumpanya ay naglalabas lang ng isang wika at nagpapasalig sa swerte.

Pinawi ng AI ang agwat

Hindi ito simpleng pagsasalin gamit ang makina. Transcreation ito. Alam ng modelo na ang "gumagana ang aming mga daga" ay hindi tumatagos sa Japan—kaya ito ay nire-rewrite bilang tugmang biro para sa kultura. Alam nito na ang $49/buwan ay magiging 月額6,980円 sa Hapon. Alam nitong kailangan ng 30% pang espasyo ang Aleman. Alam nitong nakasalin nang salamin ang layout sa Arabic at walang espasyo sa pagitan ng mga salita ang Thai.

Ituro lang ang iyong build. Makukuha ang lahat ng wika.

bunx translator-agent -s ./dist -l all

Na-inject sa mga HTML file ang lang at dir na mga atributo. Bawat lokalidad ay may CSS na may tamang font stack, taas ng linya, at direksyon ng teksto. Ang Vercel middleware ang awtomatikong humahawak ng Accept-Language detection. Ideploy nang isang beses, pagsilbihan ang buong mundo.

Patunay ito ng kakayahan

Bawat salitang binabasa mo ay dumaan sa prosesong inilalarawan ng tool na ito. Pindutin ang space para magpalit-palit ng wika. Ang bersyong Hapones ay gumagamit ng mga idyoma sa negosyo na gagamitin ng isang katutubong nagsasalita. Pumunta naman sa Arabic at panoorin ang layout na sumasalamin. Mas mahaba ang bersyong Aleman dahil ganoon talaga ito. Bawat bersyon ay parang isinulat ng isang tao mula sa kulturang iyon.

bunx translator-agent -s ./dist -l all
