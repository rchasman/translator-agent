translator-agent

A web creouse para todas e todos. Agora fala con todas e todos.

A transcreación por IA transforma o teu sitio en 71 versións locais. Non é traducido, é reescrito para cada cultura. Cada expresión adaptada, cada chiste que conecta, cada sistema de escritura respectado.

5.000 millóns de persoas están en liña. O 60 % da web está en inglés. O problema nunca foi técnico, senón económico. A localización profesional custa 10.000 € e leva un mes. Por iso a maioría das empresas publica nun só idioma e agarda o mellor.

A IA pechou a fenda

Non é tradución automática, é transcreación. O sistema sabe que "estamos a pedirllas aos nosos hámsters" non funciona en Xapón — e reescríbeo con un chiste adaptado culturalmente. Sabe que 49 €/mes é o que se di aquí. Sabe que o alemán precisa un 30 % máis de espazo. Sabe que o árabe invirte todo o deseño e que o tailandés non usa espazos entre palabras.

Apunta ao teu proxecto. Obtén todos os idiomas.

bunx translator-agent -s ./dist -l all

Os arquivos HTML reciben os atributos lang e dir automaticamente. Cada localización leva CSS con tipografía, interliñado e dirección de texto correctos. Un middleware en Vercel controla o idioma automáticamente. Publica unha vez, atende todo o mundo.

Esta páxina é a proba

Cada palabra que lees foi procesada pola ferramenta que describe. Preme espazo para cambiar de idioma. A versión xaponesa usa expresións que un nativo elixiría no ámbito empresarial. Vai ao árabe e verás o deseño invertido. A versión alemá é máis longa, como adoita ser. Cada versión léese coma se fóraa escrita por alguén desa cultura.

bunx translator-agent -s ./dist -l all
