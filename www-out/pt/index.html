translator-agent

A web foi feita para todos. Agora fala com todos.

A transcriação por IA transforma seu site em 71 versões localizadas. Não é tradução — é recriado para cada cultura. Cada expressão adaptada, cada piada no ponto, cada sistema de escrita respeitado.

5 bilhões de pessoas estão online. 60% da web está em inglês. A barreira nunca foi técnica — foi econômica. Localização profissional custa R$ 54.000,00 e demora um mês. Então a maioria das empresas publica em um único idioma e torce para dar certo.

IA fechou essa brecha

Não é tradução automática. É transcriação. O modelo sabe que "nossos hamsters estão trabalhando nisso" não faz sentido no Japão — e recria a piada para o humor local. Sabe que US$ 49/mês vira R$ 269,90 por mês. Entende que o alemão precisa de 30% mais espaço. Sabe que o árabe inverte toda a página e que no tailandês não separam as palavras.

Aponte para seu build. Tenha todos os idiomas.

bunx translator-agent -s ./dist -l all

Arquivos HTML recebem os atributos lang e dir automaticamente. Cada idioma ganha CSS com a fonte certa, espaçamento e direção do texto adequados. Um middleware do Vercel detecta Accept-Language automaticamente. Faça o deploy uma vez, sirva para o mundo todo.
