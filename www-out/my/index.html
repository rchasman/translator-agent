အင်တာနက်က တစ်ဦးချင်းစုတွေအတွက် တည်ဆောက်ထားတာပါ။ အခုကတော့ တစ်ဦးချင်းစီနဲ့ ပြောနေပြီ။

AI ဂျင်းပုံစံ ဖန်တီးခြင်းနည်းပညာက သင့်ဆိုက်ကို ပြည်တွင်းဘာသာ ၇၁ မျိုးအတွက် ပြန်လည်ဖန်တီးပေးပါတယ်။ ဘာသာပြန်တာမဟုတ်ပဲ လူ့ယဉ်ကျေးမှုအတိုင်း ပြင်ဆင်ထားတာပါ။ စကားပုံတိုင်းကို ကိုက်ညီစွာသုံးပြီး ရယ်စရာပြေပြေ အသုံးပြုတယ်၊ စာလုံးပုံစံတခုစီကိုလည်း သက်ဆိုင်ရာနည်းလမ်းနဲ့ ပြင်ဆင်ထားတယ်။

လူ ၅ ဘီလီယံကျော်ဟာ အင်တာနက်ကို အသုံးပြုနေကြတယ်။ ဝဘ်ဆိုက် ၆၀% သည် အင်္ဂလိပ်ဘာသာဖြစ်ပါတယ်။ ခြားနားမှုတာ ဝိတ္တာနည်းပညာ မဟုတ်ပဲ စီးပွားရေးဆိုင်ရာဖြစ်ပါတယ်။ အစစ်အမှန်ပြည်တွင်းဘာသာပြန်ဆိုရင် $10,000 ကုန်ကျပြီး တစ်လကြာတယ်။ ဒါကြောင့် အများဆုံးကုမ္ပဏီတွေဟာ တစ်ဘာသာသာ သုံးပြီး ရလဒ်ကို မျှော်လင့်နေကြတာပါ။

AI က ခြားနားမှုကို မဖြစ်အောင် ပြုလုပ်လိုက်ပြီ

စက်ဘာသာပြန်မဟုတ်ပါ။ ဂျင်းပုံစံ ပြန်လည်ဖန်တီးခြင်းပါ။ ဒီစက်သည် "ကျွန်ုပ်တို့ရဲ့ ဟမ်စတာတွေ ပြန်လုပ်နေကြသည်" ဆိုတာကို ဂျပန်မှာမနှစ်သက်ကြလောက်ဘူးဆိုတာ သိပြီး ယဉ်ကျေးမှုနဲ့ ကိုက်ညီတဲ့ ဟာသသစ်တစ်ခု ပြန်ရေးတယ်။ လစဉ် $49 ဟာ 月額6,980円 လို့ပြောင်းပေးတယ်။ ဂျာမနီဘာသာအတွက် နေရာ ၃၀% ပိုလိုအပ်တာကို သိတယ်။ အာရပ်ဘာသာမှာ Layout ကို လေးစားမိမိစနစ်အတိုင်း ပြောင်းလဲတယ်၊ ထိုင်းဘာသာမှာ စကားလုံးတွေ အကြား အာကာသမရှိဘူးဆိုတာလည်း သိပါတယ်။

မိမိ တည်ဆောက်ထားချက်ကို ဖော်ပြပါ။ ဘာသာစကားအကုန်လုံး ရယူပါ။

bunx translator-agent -s ./dist -l all

HTML ဖိုင်တွေအတွက် lang နှင့် dir attribute များ ထည့်သွင်းပေးပါတယ်။ ဒေသအလိုက် CSS တွေလည်း အမှန်တကယ်သုံး font stack၊ လိုင်းအမြင့် နဲ့ စာလမ်းညွှန်ချက်အတိုင်း ပါဝင်ပါတယ်။ Vercel middleware တစ်ခုက Accept-Language အလိုက် ကိုယ်တိုင် detect ပေးပါတယ်။ တစ်ကြိမ် deploy လုပ်လိုက်ရုံနဲ့ ကမ္ဘာတစ်ဝှမ်း ကို သတင်းများ ပေးပို့နိုင်ပြီ။

ဒီစာမျက်နှာက အထောက်အထားပါပဲ

သင်ဖတ်နေသော စကားလုံးတိုင်းကို ဤကိရိယာက အလုပ်လုပ်ပုံကို ဖော်ပြတဲ့ စနစ်ဖြင့် ပြန်လည်ဆောင်ရွက်ထားတာပါ။ space key ကို နှိပ်ပြီး ဘာသာစကားပေါင်းများစွာအပြောင်းအလဲ ကြည့်ရှုနိုင်ပါသည်။ ဂျပန်ဗားရှင်းက native စကားသုံးစကားပုံအသုံးအနှုန်းတွေကို သုံးထားပါတယ်။ အာရပ်ဗားရှင်းက စာမျက်နှာကို အိတ်ဖွင့် Layout အတိုင်း ပြောင်းသွားမှာကို ကြည့်ရှုနိုင်ပါသည်။ ဂျာမန်ဗားရှင်းသည် ပိုရှည်ကာ ပိုကြီးပါတယ်၊ ထို့ကြောင့် လူတိုင်းရဲ့ စကားများသဘောတူပါတယ်။ တိုင်းပြည်တစ်ခုချင်းစီနဲ့ ကိုက်ညီအောင် ကုန်ကြမ်းပါ သည်။

bunx translator-agent -s ./dist -l all
