ក្រុមហ៊ុនទាំងឡាយបង់ការទូទាត់ $0.10–0.25 ក្នុងមួយពាក្យទៅដល់ភ្នាក់ងារដើម្បីធ្វើការបែបប្រែតំបន់របស់ពួកគេ។ តំបន់បណ្តាញដែលមាន 5,000 ពាក្យទៅជា 10 ភាសា មានតម្លៃចំនួន $5,000–12,000 ហើយត្រូវការពេលវេលា 2–4 សប្តាហ៍។ រាល់ពេលដែលអ្នកផ្លាស់ប្តូរចំណងជើង ការគិតថ្លៃនឹងចាប់ផ្តើមឡើងវិញ។

## ហេតុអ្វីបានជាវាដំណើរការ

ការបែបប្រែគឺជាបញ្ហាដែលត្រូវបានដោះស្រាយរួចរាល់។ ការធ្វើបែបប្រែតំបន់មិនមែនទេ។

Google Translate អាចបែបប្រែ "Our hamsters are working on it" ទៅជាភាសាជប៉ុន។ អ្វីដែលវាមិនអាចធ្វើបានគឺ ការកំណត់អត្តសញ្ញាណថា ការលេងសើចនេះមិនប្រកាន់ក្នុងជប៉ុនទេ ហើយជំនួសវាដោយអ្វីមួយដែលប្រកាន់ — ដូចជាការបង្អាងទៅលើក្រុមវិស្វកម្មដែលធ្វើការពេញមួយយប់ ដែលទាំងសម និងកម្សាន្តក្នុងបរិបទ។

ឧបករណ៍នេះមិនបែបប្រែទេ។ វា **transcreate** — ដំណើរការដូចគ្នាដែលភ្នាក់ងារផ្សាយពាណិជ្ជកម្មគិតថ្លៃ $50,000 សម្រាប់ការសម្របសម្រួលយុទ្ធនាការឆ្លងកាត់ទីផ្សារ។ លើកលែងតែ LLM បានស្គាល់រាល់វប្បធម៌ រាល់ idiom រាល់ការកំណត់ទ្រង់ទ្រាយ។ វាដឹងថា៖

- `$49/month` ក្លាយជា `月額6,980円` ក្នុងជប៉ុន — មិនមែន "$49" ជាមួយនឹងនិមិត្តសញ្ញាយេនដាក់ទៅលើនោះទេ
- ការលេងសើចមានកម្លាំងក្នុងភាសាអង់គ្លេស ហើយស្លាប់ក្នុងភាសាជប៉ុនដែលពិធីការ
- "Drowning in paperwork" ក្លាយជា "noyade administrative" ក្នុងភាសាបារាំង — កន្សោមបារាំងពិតប្រាកដ មិនមែនការបែបប្រែពាក្យមួយបន្ទាប់ពាក្យមួយនោះទេ
- អាល្លឺម៉ង់រក្សាការលេងសើចហាំសទ័រ ព្រោះ Hamsterrad (កង់ហាំសទ័រ) គឺជា idiom អាល្លឺម៉ង់ពិតប្រាកដ
- ប្រាស៊ីលត្រូវការរបៀបសុភាពរាម ឬវានឹងស្តាប់ទៅដូចជារ៉ូបូតបានសរសេរ

ម៉ូដែលធ្វើការចាត់ថ្នាក់នូវខ្សែអក្សរនីមួយៗ។ ស្លាកយូអាយទទួលបានការបែបប្រែដោយផ្ទាល់។ សំណេររក្សាតាមទីផ្សារទទួលបានការសម្របសម្រួល។ ការលេងសើចទទួលបានការបង្កើតឡើងវិញសរុបសម្រាប់វប្បធម៌គោលដៅ។

## អ្វីដែលកើតឡើងនៅពេលអ្នកដំណើរការវា

ចង្អុលវាទៅលើលទ្ធផលសាងសង់របស់អ្នក។ វាក្លូនដើមឈើឯកសារទាំងមូលក្នុងមួយតំបន់ — បែបប្រែឯកសារអក្សរ ចម្លងទ្រព្យសម្បត្តិស្ថិតថេរ ហើយបង្កើតអ្វីៗទាំងអស់ដែលត្រូវការសម្រាប់ការដាក់ពង្រាយ៖

```
your-site/                          translations/
  index.html                          middleware.ts        ← ការកំណត់អត្តសញ្ញាណតំបន់
  about.html             →            _locales.css         ← អក្សរុក្រិតយាក្នុងមួយអក្សរ
  css/style.css                       ja/
  js/app.js                             index.html         ← lang="ja", transcreated
  images/logo.png                       about.html
                                        _locale.css        ← Noto Sans JP, 1.8 line-height
                                        css/style.css      ← copied
                                        js/app.js          ← copied
                                        images/logo.png    ← copied
                                      ar/
                                        index.html         ← lang="ar" dir="rtl"
                                        _locale.css        ← Noto Naskh Arabic, 110% font
                                        ...
                                      de/
                                        ...
```

ឯកសារ HTML នីមួយៗទទួលបាន `lang` និង `dir="rtl"` ដែលបានចាក់បញ្ចូល។ តំបន់នីមួយៗទទួលបាន CSS ជាមួយនឹងបណ្តុំអក្សរត្រឹមត្រូវ កម្ពស់បន្ទាត់ និងទិសដៅអក្សរ។ middleware របស់ Vercel ត្រូវបានបង្កើតដែលអាន `Accept-Language` ហើយសរសេរឡើងវិញទៅតំបន់ត្រឹមត្រូវ។

ដាក់ពង្រាយទៅ Vercel។ អ្នកប្រើប្រាស់ក្នុងតូក្យូឃើញភាសាជប៉ុនជាមួយ Hiragino Sans នៅ 1.8 line-height។ អ្នកប្រើប្រាស់ក្នុងគែរោឃើញអារ៉ាប់ RTL ជាមួយ Noto Naskh នៅទំហំ 110%។ អ្នកប្រើប្រាស់ក្នុងបាងកកឃើញថៃជាមួយ `word-break: keep-all` ព្រោះថៃគ្មានកន្លែងទំនេរ។ គ្មាន config។

## Caching

ការបែបប្រែគឺជាវត្ថុសិល្បៈសាងសង់។ បង្កើតនៅពេលសាងសង់ cache លទ្ធផល រំលងនៅពេលប្រភពមិនបានផ្លាស់ប្តូរ។

### Vercel

Vercel cache លទ្ធផលសាងសង់ដោយស្វ័យប្រវត្តិ។ បន្ថែម `postbuild` ហើយអ្នកបានបញ្ចប់៖

```json
{
  "scripts": {
    "build": "next build",
    "postbuild": "bunx translator-agent -s ./public -l all -o ./public"
  }
}
```

### CI

```yaml
- uses: actions/cache@v4
  id: translations
  with:
    path: translations/
    key: translations-${{ hashFiles('src/messages/**', 'public/**/*.html') }}

- if: steps.translations.outputs.cache-hit != 'true'
  run: bunx translator-agent -s ./src/messages -l all
  env:
    ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
```

ប្រភពមិនបានផ្លាស់ប្តូរ = cache hit = គ្មានការហៅ LLM = គ្មានថ្លៃ។

```
Usage: translator-agent [options]

Options:
  -s, --source <path>      ប្រភពថតឬឯកសារដើម្បីស្កេន
  -l, --locales <locales>  តំបន់គោលដៅ ផ្តាច់ដោយសញ្ញាក្បៀស ឬ "all" សម្រាប់ 71 ភាសា

  -p, --provider <name>    anthropic | openai (លំនាំដើម: "anthropic")
  -m, --model <id>         ការបដិសេធម៉ូដែល
  -c, --concurrency <n>    ការហៅ LLM ស្របគ្នាអតិបរមា (លំនាំដើម: 10)
  --api-url <url>          URL សេវាកម្មបង្ហោះ (ប្រើដោយស្វ័យប្រវត្តិនៅពេលគ្មានគ្រាប់ចាប់ API)
```
