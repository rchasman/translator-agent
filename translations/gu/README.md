# translator-agent

$10,000 ના સ્થાનિકીકરણ મુદ્દો, 90 સેકન્ડમાં ઉકેલાય.

કંપનીઓ તેમની સાઇટ્સનું સ્થાનિકીકરણ કરવા માટે એજન્સીઓને $0.10–0.25 પ્રતિ શબ્દ ચૂકવે છે. 5,000-શબ્દની સાઇટને 10 ભાષાઓમાં અનુવાદ કરવા માટે $5,000–12,000 ખર્ચ થાય છે અને 2–4 અઠવાડિયા લાગે છે. દર વાર જ્યારે તમે કોઈ હેડલાઇન બદલો છો, ત્યારે સમયનો હિસાબ ફરી શરૂ થાય છે.

આ ટૂલ તેને એક કમાન્ડમાં, 71 ભાષાઓમાં, તમારા બિલ્ડ સ્ટેપ દરમિયાન કરે છે:

```bash
bunx translator-agent -s ./dist -l all
```

કોઈ એજન્સી નહીં. કોઈ સ્પ્રેડશીટ્સ નહીં. કોઈ વેન્ડર લૉક-ઇન નહીં. કોઈ સાઇનઅપ નહીં. તમારી કી, તમારું બિલ્ડ, તમારી ભાષાઓ.

> **તમે સાક્ષી વાંચી રહ્યા છો.** આ README `bunx translator-agent -s README.md -l all` ચલાવીને અનુવાદિત થયું છે. જાઓ અને [જાપાનીઝ વર્ઝન](./translations/ja/README.md) વાંચો — તેણે માત્ર "સમયનો હિસાબ ફરી શરૂ થાય છે" નું અનુવાદ જ નથી કર્યું, પરંતુ તેને જાપાનીઝ વ્યવસાયિક શબ્દપ્રયોગથી બદલ્યું છે. [જર્મન વર્ઝન](./translations/de/README.md) 30% લાંબું છે કારણ કે જર્મન હંમેશા આવું જ હોય છે. [અરબી વર્ઝન](./translations/ar/README.md) જમણેથી ડાબે વાંચાય છે. ## આ કેમ કામ કરે છે

અનુવાદ એક ઉકેલાયેલી સમસ્યા છે. સ્થાનિકીકરણ નથી.

Google Translate "અમારા હેમ્સ્ટર્સ તેના પર કામ કરી રહ્યા છે" નું જાપાનીઝમાં અનુવાદ કરી શકે છે. જે તે કરી શકતું નથી તે છે આ જાણવું કે આ મજાક જાપાનમાં પાસે નથી, અને તેને એવી કોઈ વાતથી બદલવી જે ત્યાં પાસે — જેમ કે એન્જિનિયરિંગ ટીમ આખી રાત કામ કરી રહી છે તેનો ઉલ્લેખ, જે સાંસ્કૃતિક રીતે યોગ્ય અને સંદર્ભમાં રમૂજી બંને છે.

આ ટૂલ અનુવાદ કરતું નથી. તે **ટ્રાન્સક્રિએટ** કરે છે — એ જ પ્રક્રિયા કે જે માર્કેટમાં કેમ્પેઈન અપનાવવા માટે રેકલામ એજન્સીઓ $50,000 લે છે. સિવાય કે LLM પહેલેથી જ દરેક સંસ્કૃતિ, દરેક શબ્દપ્રયોગ, દરેક ફોર્મેટિંગ કન્વેન્શન જાણે છે. તે જાણે છે કે:

- `$49/month` જાપાનમાં `月額₹3,675` બને છે — યેન ચિહ્ન સાથે "$49" નહીં
- વ્યંગ અંગ્રેજીમાં મારી નાખે છે અને ઔપચારિક જાપાનીઝમાં મરી જાય છે
- "કાગળની ઢગલામાં ડૂબવું" ફ્રેન્ચમાં "noyade administrative" બને છે — વાસ્તવિક ફ્રેન્ચ અભિવ્યક્તિ, શબ્દે-શબ્દે અનુવાદ નહીં
- જર્મનો હેમ્સ્ટરનો મજાક રાખે છે કારણ કે Hamsterrad (હેમ્સ્ટર વ્હીલ) વાસ્તવિક જર્મન મુહાવરો છે
- બ્રાઝિલિયનોને કેઝ્યુઅલ રજિસ્ટરની જરૂર છે નહીંતર રોબોટે લખ્યું હોય તેવું લાગે છે

મોડલ દરેક સ્ટ્રિંગનું વર્ગીકરણ કરે છે. UI લેબલ્સને સીધો અનુવાદ મળે છે. માર્કેટિંગ કોપીને અપનાવવામાં આવે છે. રમૂજને ટાર્ગેટ કલ્ચર માટે સંપૂર્ણ રીતે ફરીથી બનાવવામાં આવે છે.

## જ્યારે તમે તેને ચલાવો છો ત્યારે શું થાય છે

તેને તમારા બિલ્ડ આઉટપુટ તરફ દોરો. તે લોકેલ દીઠ સમગ્ર ફાઇલ ટ્રીને ક્લોન કરે છે — ટેક્સ્ટ ફાઇલ્સનું અનુવાદ કરે છે, સ્ટેટિક એસેટ્સ કૉપિ કરે છે, અને ડિપ્લોયમેન્ટ માટે જરૂરી બધું જ જનરેટ કરે છે:

```
your-site/                          translations/
  index.html                          middleware.ts        ← લોકેલ શોધ
  about.html             →            _locales.css         ← સ્ક્રિપ્ટ દીઠ ટાઇપોગ્રાફી
  css/style.css                       ja/
  js/app.js                             index.html         ← lang="ja", ટ્રાન્સક્રિએટેડ
  images/logo.png                       about.html
                                        _locale.css        ← Noto Sans JP, 1.8 લાઇન-હાઇટ
                                        css/style.css      ← કૉપિ કરેલ
                                        js/app.js          ← કૉપિ કરેલ
                                        images/logo.png    ← કૉપિ કરેલ
                                      ar/
                                        index.html         ← lang="ar" dir="rtl"
                                        _locale.css        ← Noto Naskh Arabic, 110% ફૉન્ટ
                                        ...
                                      de/
                                        ...
```

દરેક HTML ફાઇલને `lang` અને `dir="rtl"` ઇન્જેક્ટ થાય છે. દરેક લોકેલને યોગ્ય ફૉન્ટ સ્ટેક, લાઇન-હાઇટ અને ટેક્સ્ટ દિશા સાથે CSS મળે છે. Vercel મિડલવેર જનરેટ થાય છે જે `Accept-Language` વાંચે છે અને યોગ્ય લોકેલ તરફ રીરાઇટ કરે છે.

Vercel પર ડિપ્લોય કરો. ટોક્યોનો વપરાશકર્તા 1.8 લાઇન-હાઇટ સાથે Hiragino Sans માં જાપાનીઝ જુએ છે. કાહિરાનો વપરાશકર્તા 110% સાઇઝ સાથે RTL અરબી Noto Naskh જુએ છે. બેંગકોકનો વપરાશકર્તા `word-break: keep-all` સાથે ગુજરાતી જુએ છે કારણ કે ગુજરાતીમાં સ્પેસ છે. કોઈ કૉન્ફિગ નહીં.

## તમારી પોતાની કીઓ લાવો અથવા ના લાવો

```bash
# તમારી પાસે કીઓ છે — સ્થાનિક રીતે ચાલે છે, તમે તમારા LLM પ્રદાતાને સીધું ચૂકવણી કરો છો
export ANTHROPIC_API_KEY=sk-...
bunx translator-agent -s ./dist -l all

# તમારી પાસે કીઓ નથી — બસ કામ કરે છે
# આપમેળે હોસ્ટેડ સેવાનો ઉપયોગ કરે છે
# x402 દ્વારા USDC સાથે અનુવાદ દીઠ ચૂકવણી — કોઈ સાઇનઅપ નહીં, કોઈ એકાઉન્ટ નહીં
bunx translator-agent -s ./dist -l all
```

સમાન કમાન્ડ. જો API કીઓ હાજર હોય, તો તે તમારા પ્રદાતા સાથે સ્થાનિક રીતે ચાલે છે. જો નહીં, તો તે હોસ્ટેડ API પર પહોંચે છે અને [x402](https://x402.org) દ્વારા વિનંતી દીઠ ચૂકવણી કરે છે — HTTP 402 પેમેન્ટ પ્રોટોકોલ. તમારું ક્લાયંટ Base પર USDC ચૂકવે છે, અનુવાદો પાછા મેળવે છે. કોઈ auth નહીં, કોઈ વેન્ડર સંબંધ નહીં, કોઈ ઇન્વૉઇસ નહીં.

Anthropic અને OpenAI ને સપોર્ટ કરે છે. તમે જે પણ મોડલ ઇચ્છો તે લાવો:

```bash
bunx translator-agent -s ./dist -l all -p openai -m gpt-4o
```

## દરેક સ્ક્રિપ્ટ સિસ્ટમ, હેન્ડલ થાય છે

ટૂલ માત્ર ટેક્સ્ટનું અનુવાદ જ કરતું નથી — તે જાણે છે કે દરેક લેખન પ્રણાલી કેવી રીતે રેન્ડર થાય છે:

| સ્ક્રિપ્ટ | શું બદલાય છે | કેમ |
|---|---|---|
| **અરબી, હિબ્રુ, ફાર્સી, ઉર્દૂ** | `dir="rtl"`, RTL ફૉન્ટ્સ, 110% સાઇઝ | અરબીને વાંચવા માટે મોટા ટાઇપની જરૂર છે; સમગ્ર લેઆઉટ મિરર થાય છે |
| **જાપાનીઝ, ચાઇનીઝ, કોરિયન** | CJK ફૉન્ટ સ્ટેક્સ, 1.8 લાઇન-હાઇટ | અક્ષરો ફિક્સ્ડ-વિથ ચોરસ છે; વર્ટિકલ બ્રીધિંગ રૂમની જરૂર છે |
| **હિન્દી, બંગાળી, તમિલ, તેલુગુ** | ઇન્ડિક ફૉન્ટ્સ, 1.8 લાઇન-હાઇટ | હેડસ્ટ્રોક્સ (શિરોરેખા) એ વધારાની વર્ટિકલ સ્પેસની જરૂર છે |
| **થાઈ** | `word-break: keep-all` | શબ્દો વચ્ચે કોઈ સ્પેસ નથી — બ્રાઉઝરને સ્પષ્ટ લાઇન-બ્રેક નિયમોની જરૂર છે |
| **બર્મીઝ** | 2.2 લાઇન-હાઇટ | કોઈ પણ મુખ્ય સ્ક્રિપ્ટમાં સૌથી ઉંચા ગ્લિફ્સ |
| **ખ્મેર** | 2.0 લાઇન-હાઇટ | સબસ્ક્રિપ્ટ વ્યંજન ક્લસ્ટર્સ વર્ટિકલી સ્ટેક થાય છે |

જનરેટેડ લોકેલ દીઠ CSS:

## કેશિંગ

અનુવાદો બિલ્ડ આર્ટિફેક્ટ્સ છે. બિલ્ડ સમયે જનરેટ કરો, આઉટપુટ કેશ કરો, જ્યારે સોર્સ બદલાયું ન હોય ત્યારે છોડી દો.

### Vercel

Vercel આપમેળે બિલ્ડ આઉટપુટ કેશ કરે છે. `postbuild` ઉમેરો અને તમે પૂર્ણ:

```json
{
  "scripts": {
    "build": "next build",
    "postbuild": "bunx translator-agent -s ./public -l all -o ./public"
  }
}
```

### CI

```yaml
- uses: actions/cache@v4
  id: translations
  with:
    path: translations/
    key: translations-${{ hashFiles('src/messages/**', 'public/**/*.html') }}

- if: steps.translations.outputs.cache-hit != 'true'
  run: bunx translator-agent -s ./src/messages -l all
  env:
    ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
```

સોર્સ અપરિવર્તિત = કેશ હિટ = શૂન્ય LLM કૉલ્સ = શૂન્ય ખર્ચ.
